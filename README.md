## ⚡ Pulse — Social Feed App

A modern social media feed simulation web application built using HTML, CSS, and JavaScript, featuring post creation, stories, likes, comments, reposts, notifications, analytics, hashtag trends, image uploads, and algorithmic feed behavior.

## 🚀 Features
* **Image Upload Posts** – Upload images through file selection or drag-and-drop.
* **Stories Feature** – Instagram-style stories with auto-expiry simulation.
* **Story Viewer Modal** – Full-screen story viewing experience with progress bar.
* **Like System** – Like/unlike posts dynamically.
* **Comment System** – Add comments to posts in real time.
* **Repost Feature** – Repost existing content.
* **Pin Posts** – Pin important posts to keep them at the top.
* **Delete Posts** – Remove your own posts.
* **Search System** – Search posts, hashtags, and users.
* **Hashtag Detection** – Automatically detects hashtags in posts.
* **Trending Hashtags Panel** – Displays top trending hashtags.
* **Notifications System** – Real-time notifications for likes/comments.
* **Notification Dropdown** – View and clear notifications.
